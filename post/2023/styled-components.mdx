        ## <TextSizeDown>create Global Style</TextSizeDown>

        <MarginBox lang="en">

            전역스타일 설정

            <p className="code">**CODE**</p>

            ```js
            import { createGlobalStyle } from 'styled-components';

            const GlobalStyle = createGlobalStyle`
            body {
                margin: 0;
                padding: 0;
                font-family: 'Open Sans', sans-serif;
                background: #f2f2f2;
            }

            a {
                color: #333;
                text-decoration: none;
            }
            `;


            function App() {
                return (
                <>
                    <GlobalStyle />
                </>
                );
            }

            export default App;
            ```

        </MarginBox>
    </div>
