        ## EVENT

        <MarginBox lang="en">
            1. #### dragstart

                드래그가 시작되면 발생하는 이벤트.

            2. #### dragover

                dragover 이벤트는 dragover가 설정되어있는 요소 위에서 드래그를 하고있을 때 계속 발생하는 이벤트.

                이 이벤트를 활용하면 드래그 대상의 위치를 실시간으로 추적하거나, 드래그 대상이 특정 위치에 도달했을 때 특정한 동작을 수행하는 등의 기능을 구현할 수 있습니다.

                **브라우저는 HTML 요소에 뭔가를 드롭했을 때 아무 일도 일어나지 않도록 한다.** 따라서 드롭을 위해 <Highlight>event.preventDefault</Highlight>를 호출합니다.

                <HighlightBox>

                    ### ⚡️ preventDefault

                    이벤트 객체의 메서드이다. 이벤트가 발생했을 때 브라우저의 기본 동작을 막는 역할을 한다.

                </HighlightBox>


            3. #### drop

                드래그 된 요소가 드랍됐을 때 발생하는 이벤트. drop을 위해 event.preventDefault를 호출해야한다.

        </MarginBox>

    </div>

</MainCate>

        ## <TextSizeDown>data Transfer</TextSizeDown>

        <MarginBox lang="en">

            dataTransfer 객체는 드래그 앤 드롭 중에 드래그되고 있는 데이터를 보관하기 위해 사용됩니다.

        </MarginBox>
    </div>

    <SubCate>

        <div className="sub_cate_box">
            ### Method
            <div className="sub_contents">
                <MarginBox lang="en">

                    1. #### dataTransfer.setData( )

                        드래그하는 데이터를 설정합니다. format은 데이터의 형식을 나타내는 문자열이며, data는 실제 데이터입니다.

                        지정된 타입의 데이터를 설정합니다. 타입의 데이터가 없으면 타입 목록의 마지막 항목이 새로운 형식이 되도록 타입이 마지막에 추가됩니다. 타입의 데이터가 이미 존재하면 기존 데이터가 동일한 위치에서 교체됩니다.

                        <p className="syntax">**SYNTAX**</p>

                        ```js
                        event.dataTransfer.setData('text/plain', 'data');
                        ```

                        <SyntaxBox>

                        `text`

                        dataTransfer 객체에 추가할 드래그 데이터의 타입을 나타내는 문자열. 문자열이 dataTransfer.getData 메서드를 호출할 때 사용되어, 해당 데이터를 가져올 수 있다.

                        </SyntaxBox>



                        <SyntaxBox>

                        `data`

                        dataTransfer 객체에 추가할 데이터.



                        </SyntaxBox>

                        <p className="code">**CODE**</p>

                        ```js
                        const handleDragStart = (e: React.DragEvent<HTMLLIElement>) => {
                            e.dataTransfer.setData("text", "changed");
                        };
                        ```



                    2. #### dataTransfer.getData( )

                        setData에서 첫번 째 인자로 설정한 문자열을 이용해 dataTransfer 객체에 추가된 데이터를 가져올 수 있다.

                        <p className="syntax">**SYNTAX**</p>

                        ```js
                        event.dataTransfer.getData('text');
                        ```

                        <SyntaxBox>
                        `text`

                        setData 첫 번째 인자로 설정한 문자열
                        </SyntaxBox>

                        <p className="return">**RETURN**</p>

                        setData 두 번째 인자로 설정한 데이터가 반환된다. 하지만 setData를 호출하지 않거나, 2번째 인자를 설정하지 않으면 "" 빈 스트링을 반환한다.




                </MarginBox>
            </div>
        </div>

    </SubCate>
